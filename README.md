# gst-pcapparser
Pcapparser with IPv6 support

# How to compile ?

Get meson
```
git clone https://github.com/mesonbuild/meson
```

Create build files for ninja
```
mkdir build && /path/to/meson.py build
```

# Compile 
```
ninja -C build
```
